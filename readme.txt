== ÜBER Pfadfinder Rott ==

Dieses Design wurde für die Internetseite der Rotter Pfadfinder entwickelt.

== INSTALLATION VON Pfadfinder Rott ==

- Seiten anlegen
- Kategorien anlegen
- Menü "Hauptmenü" anlegen
- Seiten hinzufügen
- Widget "Unter dem Hauptmenü" ändern
- IDs des Menüs in style.css anpassen
- Im Uploader -> server -> php -> UploadHandler.php MySQL Connection ändern
- Im Uploader -> server -> php -> UploadHandler.php -> Variablen korrigieren (s. nächster Punkt)
- Z. 596 und 602 korrigieren, falls ein anderes Wordpress-DB-Prefix als "wp_" eingesetzt wird
- Aktion "Gruppenbilder" unter Dashboard -> Bilder erstellen
- Widgets mit Textboxen füllen
